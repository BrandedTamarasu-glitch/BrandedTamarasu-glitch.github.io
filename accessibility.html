How This Site Supports Accessibility

Current accessibility practices

  • Semantic headings, landmarks, and section labels support screen-reader navigation.
  • A skip link allows keyboard users to move directly to the main content.
  • Keyboard focus states are visible across navigation, buttons, links, and controls.
  • Color contrast is tuned for readable text in light and dark themes.
  • The theme toggle exposes its current state to assistive technology.
  • External links and downloadable resume links use descriptive accessible names.
  • Resume PDFs are tagged, text-based, and kept small to preserve ATS compatibility.
  • Reduced-motion preferences are respected for users who limit animation.
  • The page includes print styles for cleaner recruiter review and offline use.

Known scope

The primary portfolio and downloadable resume files are maintained as static assets. I review the site after significant updates and keep download labels, headings, contrast, focus behavior, and mobile layout in scope for each content pass.

Feedback

If you find an accessibility issue on this site, email coryebert@outlook.com with the page, browser, assistive technology if applicable, and a short description of the issue.
